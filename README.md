# GNU/HISH - 轻量级 Linux Shell

一个用 C 语言编写的轻量级 Linux Shell，提供基本的命令行交互体验。

## 🚀 安装说明

### 快速开始

1. **下载项目**
   ```bash
   wget https://codeberg.org/master_fox/High-Shell/archive/main.zip
   unzip main.zip
   cd High-Shell

编译执行
```bash
chmod +x build.sh    # 添加执行权限
./build.sh          # 编译生成可执行文件
./highshell         # 启动 GNU/HISH Shell
##直接下载
````
如果构建失败，您可以直接下载预编译的可执行文件。
